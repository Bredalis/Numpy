<p>
  🔢 NumPy es una biblioteca fundamental para la computación científica en Python. 
  📐 Proporciona soporte para arrays multidimensionales y matrices, así como una 
  colección de funciones matemáticas de alto rendimiento para operar con estos arrays. 
  NumPy es esencial para el análisis numérico y sirve como base para muchas otras bibliotecas 
  científicas de Python, incluyendo Pandas y SciPy.  
</p>
